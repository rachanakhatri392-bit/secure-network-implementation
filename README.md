# Secure Network Implementation

## Overview

This project demonstrates the design and implementation of a secure enterprise network using Cisco Packet Tracer. The network was developed in multiple stages, beginning with a basic topology and then enhanced with internal security measures and a GRE VPN tunnel for secure communication between remote sites.

## Project Stages

### Stage 1 – Basic Network Topology
- Designed the initial enterprise network.
- Configured routers, switches, and end devices.
- Assigned IP addresses and verified connectivity.

### Stage 2 – Internal Network Security
- Improved the network by implementing internal security features.
- Enhanced network protection and organization.
- Verified secure communication between devices.

### Stage 3 – GRE VPN Implementation
- Configured a GRE VPN tunnel to connect remote networks.
- Tested communication through the VPN tunnel.
- Verified successful end-to-end connectivity.

## Technologies Used
